# Blender 4.x Import Guide (GLB) 

In Blender: File -> Import -> glTF 2.0 (.glb/.gltf).
Select exported GLB from this app.
Confirm armature and skinned clothing import under the same skeleton.
Validate unit scale (1 unit = 1 meter).
Confirm morph targets (shape keys) and test slider edits. ## Notes
Rest pose must be T-pose.
If tangents/normals warnings appear, re-export after validation fixes. ue5 import guide
